SITE-WIDE DESIGN RULES (IMPORTANT — must follow in all future work)
================================================================================

1. MAX WIDTH & PADDING
   - Content max-width: 1347px
   - Left/right padding: 16px (both sides)
   - Standard wrapper class: "max-w-[1347px] mx-auto px-[16px]"

2. BASE FONT
   - Body text: 15px, font-medium, Instrument Sans
   - Applied globally on <body> in layout.tsx

3. LOGO
   - Text: "Digilink" + a dot
   - Font: Poppins 700/800 via CSS var --font-logo
   - Dot has class "logo-dot" which runs an infinite color-cycle animation
     (red → yellow → green → blue → purple → pink → cyan → repeat, 4s loop)
   - Desktop size: text-[28px], Mobile: text-[24px]

4. BRAND BLUE COLOR: #1D349A  (used in TopBar bg, SpecialOffersTab, BottomNav hover)

5. LIQUID FILL ANIMATIONS (MANDATORY on ALL clickable/interactive elements)
   These CSS classes are defined in globals.css. ALWAYS use them instead of
   hover:bg-* classes (they conflict). Each has overflow:hidden built in.

   .liquid-fill        → Black fill from bottom-center. Use on light bg elements.
                          Hover: text turns white.
   .liquid-fill-circle → Black fill from center (circle). Use on round buttons.
                          Hover: text turns white.
   .liquid-fill-white  → White fill from bottom-center. Use on dark/black bg elements.
                          Hover: text turns dark (#0A0A0A).
   .liquid-fill-blue   → White fill from bottom. Use on blue (#1D349A) bg elements.
                          Hover: text turns #1D349A (brand blue).
   .liquid-fill-top    → White fill top-to-bottom. Use on vertical blue elements.
                          Hover: text turns #1D349A.

   Rules:
   - Every button, card, clickable box, circle gets one of these
   - Remove old hover:bg-* classes when adding liquid-fill
   - Icons inside these elements should NOT have hardcoded text-gray-* colors
     (they inherit color automatically on hover)

HEADER  (src/components/Header/)
────────────────────────────────────────────────────────────────────────────────

ARCHITECTURE:
The header is a 3-tier sticky header with scroll-aware behavior.

index.tsx — The shell/orchestrator:
  - Fixed positioned, z-50
  - Tracks: topBarHeight, headerOffset, navCollapsed, menuOpen
  - On scroll: TopBar slides up (offset = min(scrollY, topBarHeight))
    → MainHeader + BottomNav stay visible
  - When scrollY > topBarHeight: navCollapsed = true → BottomNav hides
  - A hamburger button appears in MainHeader when scrolled — clicking it
    toggles menuOpen which shows BottomNav again (dropdown style)
  - A spacer div below the header prevents content jump

TIER 1 — TopBar.tsx (blue bar, always at top):
  - bg-[#1D349A], white text
  - Left: Help Center | Find a Store | Contact links (hidden on mobile)
  - Center: "Free Express Shipping on orders $500!" announcement
  - Right: Country selector dropdown + social icons (FB, Instagram, TikTok, X)
  - Country selector:
    * 10 countries listed (Australia, Austria, Belgium, Canada, Czechia,
      Denmark, Finland, France, Germany, Hong Kong SAR)
    * Each has a CSS-rendered flag (no image files needed)
    * Default selected: Belgium (index 2)
    * Dropdown closes on outside click (useRef + mousedown listener)
    * Dropdown styled in matching blue bg with white/10 hover

TIER 2 — MainHeader.tsx (white bar, always visible when scrolled):
  - Desktop layout (lg+):
    * Left: Hamburger (appears ONLY when scrolled) + Logo
    * Center: Search bar (600px wide, rounded-full, bg-[#EDEDED])
              → Has "All Categories" dropdown button on left
              → Text input in middle
              → Search icon button on right
    * Right: "Find a store" (with location icon), "Sign in/ Register"
             (with user icon), Cart icon button (round, 40x40)
  - Mobile layout (below lg):
    * Row 1: Hamburger | Logo (centered) | Cart icon
    * Row 2: Full-width search bar (border-rounded, bg-white)
  - Hamburger uses animated 3-bar → X transition (pure CSS via transforms)
  - MenuIcon component: 3 <span> bars, animates to X when open=true

TIER 3 — BottomNav.tsx:
  - Bg white, border-bottom gray-200
  - Hidden on mobile (hidden lg:flex)
  - Height: 46px
  - Nav items (with dropdown chevrons where applicable):
    1. Shop By Categories (dropdown)
    2. Shop By Room (dropdown)
    3. Tables & Desks (dropdown)
    4. Chairs & Stools (dropdown)
    5. Pages (dropdown)
    6. Theme Features (dropdown)
    7. On Sale (NO dropdown, text-red-600 — highlighted)
  - Hover: text-[#1a1acd]
  - NOTE: No actual dropdowns are implemented yet — just the link + chevron UI

HOW TO EDIT EACH COMPONENT
================================================================================

TO CHANGE LOGO NAME:
  → MainHeader.tsx → find "Digilink" → change text

TO ADD/REMOVE NAV ITEMS:
  → BottomNav.tsx → edit the navItems array

TO CHANGE HERO SLIDES:
  → HeroBanner.tsx → edit the slides[] array
    Each slide: { id, subtitle, title, cta, href, image, bgColor }

TO CHANGE PRODUCT CARDS:
  → NewArrivals.tsx → edit the products[] array
    Each product: { category, name, price, image, colors[] }
    colors[] is array of hex strings for swatches

TO CHANGE CATEGORY CIRCLES:
  → CategoryCarousel.tsx → edit the categories[] array
    Each: { label, image, hasArrow? }

TO CHANGE FLASH SALE TIMER:
  → FlashSaleBanner.tsx → edit getTarget() function
    Current offset: 1d + 23h + 47m + 16s from page load

TO CHANGE FLASH SALE PROMO CODE:
  → FlashSaleBanner.tsx → find "Use Code: FLASH30"

TO CHANGE TOPBAR ANNOUNCEMENT TEXT:
  → TopBar.tsx → find "Free Express Shipping on orders $500!"

TO ADD MORE COUNTRIES:
  → TopBar.tsx → add to countries[] array
    Each: { name, code, flag: ReactNode (CSS-rendered flag spans) }

TO CHANGE DEFAULT COUNTRY:
  → TopBar.tsx → useState(2) → change 2 to the index of desired country

TO ADD NEW LIQUID-FILL VARIANT:
  → globals.css → add new .liquid-fill-[name]::before + :hover::before + :hover rules
    following same pattern as existing ones

TO ADD A NEW HOME SECTION:
  → Create file: src/components/Home/YourSection.tsx
  → Import and add in page.tsx after SpecialOffersTab

================================================================================
GLOBAL CSS ANIMATIONS SUMMARY (globals.css)
================================================================================

.logo-dot
  → @keyframes dot-color-cycle
  → Cycles: red→yellow→green→blue→purple→pink→cyan
  → Duration: 4s linear infinite

.liquid-fill          → ::before black bubble from bottom-center, hover expands
.liquid-fill-circle   → ::before black bubble from dead center, hover expands
.liquid-fill-white    → ::before white bubble from bottom-center (for dark bg)
.liquid-fill-blue     → ::before white bubble from bottom-center (for blue bg)
.liquid-fill-top      → ::before white bar from top, grows down (for vertical blue)

All transitions use cubic-bezier(0.4, 0, 0.2, 1) — 0.8s duration
.liquid-fill-blue is 0.5s (faster)
All use z-index: -1 on ::before, isolation: isolate on container
